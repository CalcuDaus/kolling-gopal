# Landing Page – Kopi Gerobak Listrik

## Tujuan
Membuat landing page modern untuk usaha kopi gerobak keliling menggunakan sepeda listrik. Website berfungsi sebagai media branding dan pemesanan.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Framer Motion (optional animation)
- Responsive (Mobile First)

## Style & Visual Direction
- Modern, clean, minimal
- Color : #472055
- Gradient + transparent (glassmorphism)
- Soft blur background
- Warna utama:
  - Coffee Brown (#4B2E2B)
  - Cream (#F5EDE3)
  - Accent Green/Electric (#3ECF8E)
- Background gradient:
  - Linear gradient (brown → dark green → transparent)

## Typography
- Heading: Inter / Poppins (Bold)
- Body: Inter / Sans-serif
- Ukuran heading besar, kontras tinggi

## Layout
- Full width sections
- Banyak whitespace
- Card dengan background semi-transparent
- Rounded corner (xl)
- Shadow halus

## Section Detail

### Hero Section
- Headline:
  “Kopi Enak, Datang ke Kamu”

  “Kopi gerobak keliling berbasis sepeda listrik — ramah lingkungan & fresh brew”
